# Contributing

The source code is open source software released under the [MIT license](./LICENSE).

## External contributions cannot be accepted currently

At this point in time, source code is released as open source software based on the [EMBAG](https://www.fedlex.admin.ch/eli/cc/2023/682/de) law.

We do not currently accept external contributions. Please refrain from providing code changes or pull requests as we are unable to accept them.

## Reporting Security Vulnerabilities

If you would like to report a potential security issue, please follow the procedure described in [SECURITY.md](./SECURITY.md).
